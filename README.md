# Distributed-TensorFlow-Example
About a cluster of TensorFlow servers, and how to distribute a computation graph across that cluster

## Requirements
- Python 3.5.2
- TensorFlow >= 1.3.0

## Contents
* Logistic-Regression
* CIFAR-10
